About this App

This site was created as part of my 4th year honours project at Glasgow University. It may be rough around the edges in a few sections but I hope you can still enjoy the main content and features.

The main feature of this site is the ability to simulate a Turing Tumble board game but also includes puzzle making and sharing.

Tutorial

This app runs entirely in the browser, so requires no install. Use the side navigation bar to the left to navigate to other parts of the app.

If you are unfamiliar with the app and the Turing Tumble board game visit the tutorial section here!

Plain Board

The first place to start if you've read the tutorial is with a plain board which will have a few examples of more complicated Turing Tumble boards, plus the ability to make your own board!

Make a board from your imagination or visit some of the featured boards below for inspiration.
