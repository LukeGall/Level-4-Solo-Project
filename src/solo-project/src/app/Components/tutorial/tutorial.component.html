Site Instructions

This virtual turing tumble runs in the browser and there is nothing for you to install. Find the sidenav to the left of the screen to go to the different pages of the site. These include

  • Home which gives a brief introduction to the site and why it was made.
  • The Plain Board page will give a plain board which will allow you to play with a Turing Tumble simulation.
  • The Original Puzzles page will give a list of puzzles to play which will be explained in a different tab.
  • Online Puzzles will require you to log in to access and play other users created puzzles.
  • Finally the Create Puzzle page will let you make your own puzzles for other users to play.


    • The site has two themes, light and dark, which can be changed on the fly using the button on the top header bar.

    Turing Tumble Instructions

    A Turing Tumble board involves blue and red marbles, shown at the top left and right corner of the board respectively. These marbles travel down the board interacting with various pieces which mimic parts of a computer circuit. There are 2 triggers at the bottom of the board which start the marble down the board. The marble can only travel down the board along routes of pieces, if not it will 'fall'. When a marble gets to the bottom of the board it will release another blue or red marble depending on the side the marble gets to. It will also be 'collected' in which the board will store the marbles that have reached the bottom. An example is shown below in which 2 blue marbles were collected, then 3 red and finally 5 blues. The main objective of the game is to place a combination of pieces onto the board to create a mechanical computer that will output a certain marble pattern.


    The number of coloured marbles can be found in the top left and right of the board, the amount of marbles can be increased / decreased using the 'plus' or 'minus' buttons. The blue marbles drop in from the blue arrow at the top of the board while the red marbles drop in from the red arrow. In the board below you'll see different 'slots' on the board, Component Slots and Pins. Component slots allows all 6 components to be placed, while Pins only allow Gears to be placed.

    Component Slots and Pins

    Component Slot Pin

    All pieces can be fitted to the Component Slots but only Gears can be attached to the Pins. The Pieces are required to transport the marble to the bottom of the board and each have different uses. The currently held piece will be shown by the colour change on the selection sidebar when clicked. You can also hold 'Delete Piece' to remove any pieces from the board.

    A puzzle in Virtual Turing Tumble will come with a

  • Title
  • Description (objective) that will give an overview of the puzzle and describe how to win, this can sometimes entail an objective about the board itself not the marble output e.g. intercept a ball.
  • Starting Board set up are the initial pieces on the board that help give hints towards the puzzle. These pieces can't be replaced.
  • Required Output this the output needed to 'win' the puzzle. Some puzzles won't have a required output as it will be focused more on the board state e.g. move 2 bits.
  • Available parts there parts are the only ones that can be used to meet the objective, no more can be used and it is likely that all will be needed.


    • To play a puzzle, go to either the Original Puzzles page which contain puzzles created using the puzzle book from the physical Turing Tumble or visit Online Puzzles to play puzzles made by other users of the site. On both pages you will see a list of puzzles that you can filter by difficulty and when selecting a puzzle drop down arrow which gives details about the puzzle, including it's description the starting board set up and the output required to complete the puzzle.

    Making a puzzle

    A puzzle can be made from the Create Puzzle page an involves three steps

  • First, make the starting set up for your board and confirm the set up once it is finished. Use the 'Confirm set up' button when ready.
  • Second, add pieces to the board which will act as the solution to your puzzle, increase the necessary marbles and play the puzzle until the collected marbles at the bottom equals your required output, it is okay to leave this blank if your puzzle doesn't have one. When ready click the 'Confirm set up' button.
  • Lastly, after confirming the solution and final output you will be taken to a form to fill out the title, description and difficulty of your puzzle.
